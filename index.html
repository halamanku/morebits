Apa itu VPN Remote?

VPN REMOTE merupakan layanan VPN khusus untuk remote MikroTik dari luar jaringan walaupun anda tidak mempunyai IP Public. Sangat cocok bagi anda yang sering berada di luar jaringan namun ingin tetap dapat memantau jaringan anda. Selain router MikroTik, VPN REMOTE juga dapat digunakan untuk memantau perangkat di bawahnya seperti router ataupun access point, serta berbagai fungsi spesial lainnya.

Fungsi VPN Remote

REMOTE MIKROTIK

VPN REMOTE utamanya ditujukan untuk remote mikrotik dari luar jaringan. Bisa menggunakan winbox, webfig, dan lain-lain.

REMOTE PERANGKAT LAIN

VPN REMOTE dapat digunakan untuk remote perangkat di belakang Mikrotik, seperti access point, router, web server, dan lain-lain.

TUNNEL API MIKROTIK

VPN REMOTE dapat digunakan sebagai tunnel aplikasi yang menggunakan API MikroTik seperti Mikhmon, Mikbotam, dan lain-lain.

MENGGANTI NAMA ISP

VPN REMOTE dapat digunakan untuk mengganti (menyembunyikan) nama ISP dan IP publik yang muncul pada aplikasi speedtest.
